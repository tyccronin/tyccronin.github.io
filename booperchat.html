Return Home


This app is my favourite yet! Made with React and the incredible API Chat Engine. After a user is created through Chat Engine's backend through the account page on their website, that user can log into via the login screen with their assigned username and password. Then, simply start chatting! The app includes functionality such as read receipts, avatars, and the ability to not only send messages, but also images. There is no sign-out functionality at the moment, so you'll have to clear your local storage in the browser to log out.
